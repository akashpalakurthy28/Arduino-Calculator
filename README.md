# Arduino-Calculator
Arduino Calculator using Arduino uno

## Overview
This project implements a basic calculator using an Arduino uno, a 4x4 keypad, and a 16x2 LCD display. It performs addition, subtraction, multiplication, and division.

# Features
- Addition
- subtraction
- multiplication
- Division
